== Installation ==
 
如何安装 SEO External Link.

方法一：

1. 点击插件介绍页的 Download 下载 SEO External Link 插件安装包
2. 登录 WordPress 站点后台, 点击后台左侧菜单的插件项
3. 依次点击 安装插件 - 上传插件 - 选择文件 - 现在安装
4. 安装完成后, 点击启用即可

方法二：

1. 登录 WordPress 站点后台, 点击后台左侧菜单的插件项
2. 然后, 点击安装插件
3. 在右边搜索框中输入 SEO External Link 按回车键进行搜索
4. 选择 SEO External Link 插件(注意,作者是艾码汇) 点击现在安装
5. 安装完成后, 点击启用即可

== Frequently Asked Questions ==

1. 如何设置不需要跳转页面域名
点击左侧菜单 - 设置 - 常规，在右边找到安全链接设置，添加不需要跳转页面的域名即可

2. 设置了不需要跳转页面域名，为何仍然出现跳转页面
请注意设置是否正确，每个域名为一行，而且域名形式如：www.imahui.com，不需要添加访问协议
